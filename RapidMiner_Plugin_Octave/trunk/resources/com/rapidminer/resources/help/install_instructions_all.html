1. Installation Guide (all platforms)

For using the Octave Extension of RapidMiner, you need to have Octave installed on your computer (http://www.gnu.org/software/octave/). You will need Octave version 3.6.1.

  1. After you have installed a proper version of Octave (3.6.1 is supported currently), you may want to add it to the PATH environment variable of your OS - in such case you don't need to configure anything else, simply restart Rapidminer when Octave is in the PATH.
  2. If for some reason you don't want to add it to the PATH, you will need to specify the Octave executable explicitely using the Select Octave Engine button below. The path to Octave executable is typically C:\Octave\Octave3.6.1_gcc4.6.2\bin\octave.exe(windows) or octave (linux).
  3. Once you have changed your PATH variable or selected the Octave executable file manually, exit and restart Rapidminer manually.

2. Using Octave operators

The Execute Octave Script operator allows to execute various Octave scripts as well as consuming and delivering ExampleSets. Please refer to the operator documentation for more details.
